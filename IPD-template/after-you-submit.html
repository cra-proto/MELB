After you submit

This section bridges the gap between the user completing their action and the outcome. It tells the user what they should do (or stop doing) immediately after submitting.

On this page

  • What to include
  • Guidance
  • Recommended headings

What to include

  • Confirmation of what just happened (“Your application has been submitted” or “Your return has been received”)
  • Whether the user should expect a confirmation number, email, or reference ID
  • Whether the user needs to keep any records, receipts, or copies of what they submitted
  • Any immediate next steps the user must take (for example, mailing supporting documents separately)

Guidance

  • Keep this section short and reassuring
  • If there is nothing the user needs to do after submitting, say so explicitly: “You do not need to do anything else.”
  • If the user submitted by mail, tell them not to send a duplicate

Recommended headings

  • After you claim
  • After you submit
  • After you file
