Version inconsistencies

Font Awesome version drift

Font Awesome is loaded at three different versions across different pages: v5.5.0 (most pages), v5.8.1 (404 page), and v5.15.4 (some newer pages). This can cause inconsistent icon rendering. New pages should use v5.15.4 for consistency going forward.

Unpinned WET-BOEW

WET-BOEW core JS and GCWeb theme CSS are loaded from wet-boew.github.io/themes-dist/ without a version pin. This means the site automatically gets the latest release, which could occasionally cause breaking changes. Consider pinning to a specific release tag.
