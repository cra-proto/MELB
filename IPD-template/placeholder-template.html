Placeholder template

Copy this structure when starting a new page. Delete sections that do not apply; fill in sections that do. Every deleted section should be a deliberate decision, not an oversight.

Use the section applicability table to determine which sections are required, recommended, or optional for your page type.

Template

<h1 property="name" id="wb-cont" class="gc-thickline">
  [Action verb] + [the thing]
</h1>

<p>[1-2 sentence description of what this page
covers and who it is for.]</p>

<h2 class="h3">On this page</h2>
<ul>
  <li><a href="#before">Before you start</a></li>
