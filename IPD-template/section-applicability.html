Section applicability by page type

Not every “How to do X” page needs all 14 sections. Use this table to determine which sections are typically required, recommended, or optional depending on what the page is about.

Section applicability by page type
Section Apply for a benefit File or submit Register or enrol Claim a credit or deduction Check or update info

Contact infoRecommendedRecommendedRecommendedRecommendedRecommended

Key

  • Required – Users will almost certainly need this information. Omitting it will likely generate calls or create confusion.
  • Recommended – Most pages of this type should include this. Omit only if genuinely not applicable.
  • Optional – Include if relevant to the specific service. Not every page of this type will need it.
