Step-by-step instructions

The core of the page. This section walks the user through the process of actually doing the thing.

On this page

  • What to include
  • Guidance
  • Example pattern
  • Recommended headings

What to include

  • Numbered steps in chronological order
  • What to do at each step
  • Where to go (which form, which portal, which phone number)
  • What to enter, select, or attach
  • Decision points: what to do if the user’s situation varies

Guidance

  • Use the WET ordered-step list pattern (lst-stps ld-zr) for consistent styling
  • Start each step with an action verb: “Sign in to...”, “Select...”, “Enter...”, “Attach...”
  • Keep each step to one action when possible. If a step has sub-actions, use sub-steps
  • Include screenshots or annotated images only when the interface is genuinely confusing; they go stale quickly
  • When there are multiple ways to complete the task (online, by mail, by phone), present the most-used method first and use expand/collapse or tabs for alternatives
  • Do not embed eligibility or background information inside the steps; it belongs in its own section
  • Limit the total number of steps — if you have more than 8–10, consider the subway pattern

Example pattern

  1. Sign in to My Account

    Go to My Account and sign in with your CRA credentials.

  2. Select the service

    From the main menu, select [Service name].

  3. Complete and submit the form

    Fill in the required fields and select Submit.
