# cra-writing

Writes and edits CRA web content following the User-Centred Design Guide (UCDG) writing standards.

## What it does

Applies CRA's official writing rules when creating or reviewing web content for Canada.ca and CRA pages. Covers tone, plain language, punctuation, formatting, terminology, and content structure.

## When to use

- Writing or rewriting web page text for CRA or Canada.ca
- Editing drafts for plain language, tone, or style compliance
- Reviewing content against UCDG writing standards
- Questions about CRA content style rules

## Key rules enforced

- Grade 8 reading level (Flesch-Kincaid)
- Sentences under 20 words, paragraphs under 3 sentences
- Active voice, second person ("you"/"your"), no FAQs
- CRA-specific naming, obligation language (must/need to/may), and formatting conventions
- Punctuation, emphasis, numbers, dates, and contact information standards

## Source

Rules extracted from the `en/writing/` section of the [CRA User-Centred Design Guide](https://github.com/cra-proto/design).
