# gcweb-doormats

Generates GCWeb doormat navigation patterns for Canada.ca and CRA web pages.

## What it does

Produces accessible, standards-compliant doormat HTML — sets of linked tiles with short descriptions used as the primary navigation pattern on Government of Canada pages.

## When to use

- Creating doormats, navigation tiles, or link-and-description blocks
- Building GCWeb index pages, topic pages, institutional landing pages, or subway index pages
- Any page that needs a set of linked sections with summaries

## Key rules enforced

- Link titles: max 75 characters (aim for 45)
- Descriptions: max 120 characters, no trailing punctuation, no generic text
- Parallel grammatical structure across all doormats on a page
- Correct WET/GCWeb HTML structure (`wb-eqht row`, `col-lg-4 col-md-6`)
