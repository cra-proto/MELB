# gcweb-links

Generates and reviews GCWeb link patterns for Canada.ca and CRA web pages.

## What it does

Produces accessible, standards-compliant hyperlinks with descriptive link text, proper placement, and correct design variants including button links, rescue links, email links, and vanity URLs.

## When to use

- Creating or reviewing hyperlinks on a CRA or Canada.ca page
- Implementing rescue links ("You may be looking for")
- Writing email links with full address as visible text
- Adding button links or download links
- Checking link text accessibility and placement

## Key rules enforced

- Descriptive link text (no "click here" or "read more")
- Links placed at end of sentences, not mid-sentence
- Related links introduced with a verb and colon (never "see")
- Email addresses written in full as visible link text
- Maximum one button link per page
- Same tab by default; new tab only when same-tab interrupts the task

## Source

Rules extracted from the `en/design-systems/gcweb/links/` section of the [CRA User-Centred Design Guide](https://github.com/cra-proto/design).
