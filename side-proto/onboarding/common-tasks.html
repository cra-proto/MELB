Common tasks cheat sheet

Quick-reference instructions for the tasks you will do most often.

On this page

  • Add a new content page
  • Add a top-level nav item
  • Add a changelog entry
  • Add a GCWeb component page
  • Add a new activity
  • Create a new banner image

Add a new content page

  1. Copy an existing page at the same directory depth
  2. Rename using kebab-case
  3. Update: lang, title, dcterms, language toggle, breadcrumbs, all relative paths, content, date modified
  4. Create the French counterpart
  5. Cross-link the language toggles in both files

Add a top-level nav item

  1. Edit assets/includes/menu-include-en.html
  2. Add <li> to the desktop <ul class="list-inline menu">
  3. Add the same link to the mobile overlay section (#mb-pnl)
  4. Repeat in menu-include-fr.html

Add a changelog entry

  1. Open assets/files/changes/whats_new_ucdg.json
  2. Add a new object at the beginning of the array
  3. Fill in all bilingual fields (_en and _fr)
  4. Commit — the changes page and home page update automatically

Add a GCWeb component page

  1. Create thumbnail pair: assets/images/banners/gcweb/[name]-en.jpg and [name]-fr.jpg
  2. Add a <tr> row to <tbody> in en/design-systems/gcweb/index.html and the French equivalent
  3. Create the detail page at en/design-systems/gcweb/[name]/index.html
  4. Follow the standard component page structure: banner, intro, “On this page”, when to use, how to implement, design options, accessibility
  5. Add a changelog entry

Add a new activity

  1. Copy en/activities/usability-testing.html (the best example of a complete activity page)
  2. Follow the standard sections: When to use, Who is involved, How to do it (steps), Deliverables, Tools, References
  3. Add a row to the filterable table in en/activities/index.html
  4. Link the activity from the appropriate process phase page

Create a new banner image

  1. Create 4 PNG files with transparent backgrounds:
    • [name]-991.png (990 × 800, image area 990 × 250)
    • [name]-1400.png (1400 × 800, image area 1400 × 300)
    • [name]-1700.png (1700 × 800, image area 1700 × 300)
    • [name]-1900.png (1900 × 800, image area 1900 × 200)
  2. Save to assets/images/banners/
  3. Reference them in the page using data-bgimg-srcset
