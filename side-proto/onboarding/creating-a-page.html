How to create a new page

There is no template engine. Creating a new page means copying an existing page at the same directory depth and updating everything by hand. Follow these steps exactly.

On this page

  • Choose your starting point

  • New page checklist

Choose your starting point

Copy a page that is at the same directory depth as where your new page will live. This ensures the relative paths to CSS, JS, and images are correct.

Which page to copy based on target location
Your new page lives inCopy this pageWhy
/en/writing/en/writing/tone.htmlSame depth, same section, has “On this page” nav and sidebar
/en/activities/en/activities/usability-testing.htmlUses the activity page template with all standard sections
/en/design-systems/gcweb/[component]/en/design-systems/gcweb/alerts/index.htmlComponent page with example tabs and clipboard
/en/blog/en/blog/2024-05-accessible-content.htmlBlog post format

Do not copy a page from a different depth. A page at /en/writing/tone.html uses ../../assets/ for asset paths. A page at /en/design-systems/gcweb/alerts/index.html uses ../../../../assets/. Getting this wrong breaks CSS, JS, and images silently.

Step-by-step process

  1. Copy the source page

    Duplicate the HTML file and rename it using kebab-case (for example, my-new-topic.html).

  2. Update the <html lang> attribute

    Set to en or fr. This controls which menu and footer ucdg.js loads.

  3. Update the <title> tag

    Follow the pattern: [Page title] - [Section] - User-Centred Design Guide - Canada.ca.

  4. Update dcterms metadata

    Set dcterms.title, dcterms.description, dcterms.issued, and dcterms.modified.

  5. Update the language toggle link

    Point the <a lang="fr"> (or <a lang="en">) to the corresponding page in the other language. This is a relative path.

  6. Update breadcrumbs

    Manually edit the <ol class="breadcrumb"> to reflect the page’s location in the site hierarchy. Each level is a manual <li> entry.

  7. Verify all relative paths

    Check every src and href that points to /assets/. Count the ../ segments. They must match the directory depth of your new page.

  8. Replace the content

    Write your page content following the section’s content pattern (see the page type guidance in the How to create a page guide).

  9. Update the date modified

    Set the dateModified value at the bottom of the page.

  10. Create the French counterpart

    Copy the new English page into the corresponding French directory, translate the content, and update the language toggle links in both directions.
