ucdg-theme.css in detail

This file does two things that define the site’s visual identity:

Menu-inverse theme (lines 1–118)

Overrides WET-BOEW’s default dark blue menu to create a light, transparent background. Key choices:

  • Menu background: transparent
  • Link text: #333 (dark grey), 20px font-size
  • Active/hover: #f5f5f5 background with a 5px bottom border
  • Open submenu: #4d4d4d dark background with white text
  • Mobile overlay: dark background with static positioning

Step list component (lines 119–288)

Creates the numbered circle markers used on ordered lists across the site. Two variants:

  • lst-stps — default with circular numbered markers (border colour #26374a)
  • lst-stps ld-zr — leading-zero variant (01, 02, 03) with a left-border-only style
  • stps-strpd — adds alternating grey/white row backgrounds

and-or.css in detail

A CRA-specific component for showing “and” or “or” relationships between content blocks. Uses CSS content property with language-aware selectors:

  • html:lang(en) .cnjnctn-type-and → displays “and”
  • html:lang(fr) .cnjnctn-type-and → displays “et”
  • html:lang(en) .cnjnctn-type-or → displays “or”
  • html:lang(fr) .cnjnctn-type-or → displays “ou”

No CSS variables are used anywhere. All values are hardcoded. The colour palette is small: #333, #4d4d4d, #6f6f6f, #f5f5f5, #26374a, #e3e3e3, #d6e9c6.
